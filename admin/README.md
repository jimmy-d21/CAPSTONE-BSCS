# Ribshack HQ — Admin Dashboard

React JSX admin dashboard for Ribshack Philippines.

## Setup & Run

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Pages

- **Global Analytics** — KPIs, regional performance, charts
- **Branch Management** — Add/edit/delete branches, view store dashboards
- **Store Dashboard** — Per-branch revenue, orders, inventory alerts
- **Request Center** — Approve or decline branch restock requests
- **Product Catalog** — Add/edit/delete menu items, toggle Unli-Rice
- **Finance & VAT** — Monthly P&L, branch financials, BIR VAT reports

## Tech Stack

- React 18 + Vite
- React Router v7
- Tailwind CSS v4
- shadcn/ui (Radix UI)
- Recharts
- Sonner (toasts)
